# Roadmap

## Phase 1: Strong Foundations
- Revise JavaScript fundamentals
- Practice problem solving
- Build small but complete projects

## Phase 2: Project Building
- Create mini projects using JavaScript
- Focus on clean code and understanding
- Learn how to debug and improve existing code

## Phase 3: Open Source Entry
- Learn how open-source projects work
- Start with documentation and small fixes
- Make first pull requests

## Phase 4: Consistent Contribution
- Contribute regularly to selected projects
- Interact with maintainers and community
- Improve code quality and communication

## Phase 5: GSoC Preparation
- Shortlist suitable organizations
- Understand project ideas deeply
- Prepare and refine GSoC proposal
